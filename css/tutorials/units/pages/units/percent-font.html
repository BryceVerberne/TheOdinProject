Given that everything has the same font, everything is modified to have the same value. Each example is effectively getting 200% of the root font-size.

As you can see, as we change the parent to have a font-size of 30px, `2rem` stays that same since it is staying true to the root font-size. However, both `2em` and `200%` change to have a font-size of 60px since they both double the font's size.

Using percentages with fonts do the exact same thing as how `em` interacts with fonts.
