Inside the `parent` we have defined a new font size. This means that `em` is going to change according to the parent's font size. However, given that `rem` only takes into consideration the root font size, it does not change, which can be seen when comparing it to the example above.

Given that the 1 in-front of `em` and `rem` stands for 100% the font size, by putting a 2 in-front of them means that we want 200% the font size. Therefore, if the root font size is 16px, `2rem` will have a font size of 32px, and if the parent font size is 30px, `2em` will have a font size of 60px.

By using `em` with shapes, as we scale up the font size, so does the size of the box. This is why the third box is so large. We scaled the font up to 30px, meaning that our box is accounting for this new font size. However, `rem` stays the same since it remains true to the root font size.

Ultimately, `em` is great since it provides scaling capabilities and `rem` is useful for when you want to set a font to the base size, even if you are unsure of what that is.
